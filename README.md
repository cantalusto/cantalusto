Right now, I’m basically living in the web dev universe 😄  

On the backend side, I’m working with **PHP** and **Laravel**, using **Artisan** and **Laravel Sail** to keep everything running smoothly inside **Docker** containers. It makes my environment clean, organized, and way easier to manage (no more “it works on my machine” moments).

For the frontend, I’m jumping between **Vue** and **React**, and when I want extra performance and a solid structure, I bring **Next.js** into the mix. Styling? That’s all **Tailwind CSS** — fast, flexible, and perfect for building clean, modern interfaces without wasting time.
